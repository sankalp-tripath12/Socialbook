SocialBook is a simple, colorful, and user-friendly social media website inspired by Facebook. It includes pages for connecting with friends, sharing posts, and exploring content.

1.Features

Home Page – Overview of updates and posts.

Profile Page – View and edit user information.

Pages – Explore different interest pages.

Feed – See posts from friends and pages.

Friends – Manage your friend list.

Groups – Join and interact in groups.

Fully responsive and visually attractive design.
